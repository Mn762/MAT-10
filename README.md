# MAT 10

## Setup

- install python version >=3.8
- install git from https://git-scm.com/downloads
- fork this repository
- run git clone

## How to run the scripts

Run the script from inside the experiment's folder


- `cd exp1-casting/`
- `python exp1-casting.py`
